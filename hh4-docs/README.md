# HH4 Platform Documentation Generator

## Prerequisites
- Python 3.9+
- pip install -r requirements.txt

## Generate Word Document
```bash
python generate_word.py
# Output: HH4_Platform_Solution_Architecture.docx
```

## Generate PowerPoint Presentation
```bash
python generate_pptx.py
# Output: HH4_Platform_Presentation.pptx
```

## Notes
- Both scripts are standalone, no external data files needed
- All content is embedded in the scripts
- Run time: ~10-30 seconds each
